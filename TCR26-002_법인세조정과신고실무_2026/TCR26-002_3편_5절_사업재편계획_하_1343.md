## 작성요령 V - 세액감면신청서

[별지 제35호 서식] (2017. 3. 17. 개정)

세액감면신청서의 주요 기재 항목은 다음과 같다.

| 항목 | 설명 |
|------|------|
| ①법인명 | 신청법인의 회사명 |
| ②사업자등록번호 | 신청법인의 사업자등록번호 |
| ③대표자 성명 | 대표자의 이름 |
| ④법인등록번호 | 대표자 생년월일 또는 법인등록번호 |
| ⑤사업장(본점)소재지 | 본점 주소 및 전화번호 |
| ⑥업종 | 업종코드 및 업종명 |
| ⑦사업개시일자 | 사업 개시일 |
| ⑧사업연도 | 과세 대상 사업연도 |

세액감면신청서는 세무서장에 제출하며, 첨부서류로는 증여계약서 사본 1부를 제출한다.

## 5. 사업재편계획에 따른 기업의 채무면제익에 대한 과세특례 - 서식

### 작성요령 I - 사업재편계획서(사업재편계획에 따른 채무면제익)

[별지 제70호의 15 서식] (2022. 3. 18. 개정)

#### 작성 대상 및 방법

사업재편계획에 따른 채무면제익으로 손금산입을 신청하고자 하는 내국법인이 작성한다.

#### 기재 항목

1. 대상법인 인적사항

| 항목 | 내용 |
|------|------|
| 법인명 | 채무면제익을 계상하는 법인 |
| 사업자등록번호 | 법인의 사업자등록번호 |
| 대표자 성명 | 법인 대표자 성명 |
| 법인등록번호 | 법인등록번호 |
| 사업장(본점)소재지 | 본점 소재지 및 연락처 |
| 업종 | 업종코드 및 명칭 |
| 사업개시일자 | 사업 개시일 |
| 사업연도 | 해당 과세연도의 시작일과 종료일 |
| 사업재편계획승인권자 | 사업재편계획 승인 기관명 |

2. 채무명세 및 면제 계획

금융채권자로부터 받은 채무의 세부 내역을 기재하며, 다음 항목을 포함한다.

| 항목 | 설명 |
|------|------|
| 금융채권자명 | 채무를 면제한 금융기관 또는 채권자명 |
| 사업자등록번호 | 금융채권자의 사업자등록번호 |
| 차입일자 | 채무 발생 일자 |
| 채무금액 | 원래 채무액 |
| 면제예정일자 | 채무면제 계획 일자 |
| 면제예정금액 | 면제 예정액 |

제출 시기와 방법: 조세특례제한법 시행령 제116조의 33 제7항에 따라 사업재편계획서를 제출한다. 제출인과 확인자의 서명 또는 인장을 받아 세무서장에 제출한다.

### 작성요령 II - 사업재편계획이행보고서(사업재편계획에 따른 채무면제익)

[별지 제70호의 16 서식] (2022. 3. 18. 개정)

#### 작성 목적

사업재편계획에 따른 채무면제익이 계획대로 이행되었음을 보고하기 위한 서식이다.

#### 기재 항목

1. 대상법인 인적사항

사업재편계획서에 기재한 것과 동일한 법인의 인적사항을 기재한다.

| 항목 | 설명 |
|------|------|
| 법인명, 사업자등록번호 | 채무면제익 계상 법인 정보 |
| 대표자 성명, 법인등록번호 | 대표자 정보 |
| 사업장(본점)소재지 | 주소 및 전화번호 |
| 업종, 사업개시일자 | 업종 및 개시일 |
| 사업연도 | 과세기간 |
| 사업재편계획승인권자 | 승인 기관명 |

2. 사업재편 이행 보고내용

금융채권자의 채무 면제 이행 현황을 기재한다. 면제가 계획대로 이루어졌는지 확인하고, 실제 면제 일자와 면제액을 기록한다.

제출 시기: 세무서장에 제출하며, 제출인과 확인자의 서명을 받는다.

### 작성요령 III - 채무면제명세서

[별지 제38호 서식] (2023. 3. 20. 개정)

#### 작성 대상

조세특례제한법 제121조의 29 제3항을 적용받으려는 금융채권자는 채무를 면제받은 내국법인 중 채무면제금액을 손금에 산입하고자 하는 법인별로 작성한다.

#### 기재 항목

1. 채무를 면제받은 법인 정보

| 항목 | 내용 |
|------|------|
| ①법인명 | 채무면제를 받는 법인 |
| ②사업자등록번호 | 법인의 사업자등록번호 |
| ③대표자 성명 | 대표자명 |
| ④업종 | 업종코드 및 명칭 |
| ⑤본점소재지 | 본점 주소 |
| ⑥사업연도 | 과세연도 |
| ⑦정리계획인가의 결정일 등 | 법정관리 등 판단 기준이 되는 날짜 |
| ⑧특수관계 여부 | 금융채권자와의 특수관계 확인 |

2. 채무를 면제한 금융채권자 정보

| 항목 | 내용 |
|------|------|
| ⑨법인명 | 면제를 제공한 금융기관 등 |
| ©대표자 성명 | 금융채권자의 대표자 정보 |
| ©본점소재지 | 금융채권자의 주소 |

3. 채무면제 내역

| 항목 | 설명 |
|------|------|
| ©채무면제일 | 실제 면제가 이루어진 일자 |
| ⑩채무면제금액 | 면제된 채무액 |

4. 출자전환 채무면제 내용

채무를 주식으로 전환하는 경우 다음을 기재한다.

| 항목 | 내용 |
|------|------|
| ©출자전환일 | 채무가 주식으로 전환된 날짜 |
| ⑩채무금액 | 전환 대상 채무액 |
| ⑩1주당 주식의 발행가액 | 발행 주식의 1주당 가액 |

제출 시기와 방법: 조세특례제한법 시행령 제116조의 33 제8항에 따라 채무면제명세서를 제출한다. 첨부서류로는 정리계획인가, 화의인가, 강제화의인가 결정문 등을 제출한다.

## 6. 기업 간 주식 등의 교환에 대한 과세특례 - 서식

### 작성요령 I - 사업재편계획서(기업 간 주식 등의 교환)

[별지 제70호의 17 서식] (2022. 3. 18. 개정)

#### 작성 개요

주식 교환을 통해 기업 간 주식을 양도·양수하는 법인이 사업재편계획서를 작성한다.

#### 기재 항목

1. 교환대상법인(주식을 양도하는 법인)

| 항목 | 설명 |
|------|------|
| 법인명 | 양도 법인명 |
| 사업자등록번호 | 양도 법인의 사업자등록번호 |
| 대표자 성명 | 대표자명 |
| 법인등록번호 | 법인등록번호 |
| 사업장(본점)소재지 | 본점 주소 및 전화번호 |
| 업종 | 업종 |
| 사업개시일자 | 개시일 |
| 사업연도 | 과세연도의 시작일과 종료일 |
| 사업재편계획승인권자 | 승인 기관명 |

2. 교환양수법인(주식을 양수하는 법인)

| 항목 | 설명 |
|------|------|
| 법인명 | 양수 법인명 |
| 사업자등록번호 | 양수 법인의 사업자등록번호 |
| 대표자성명 | 대표자명 |
| 법인등록번호 | 법인등록번호 |
| 사업장(본점)소재지 | 본점 주소 및 전화번호 |
| 업종 | 업종 |
| 사업개시일자 | 개시일 |

3. 주식양도·양수 계획

주주별로 보유 주식 현황과 양도·양수 계획을 다음과 같이 기재한다.

| 항목 | 설명 |
|------|------|
| 주주명 | 주주의 성명 또는 법인명 |
| 생년월일 또는 법인등록번호 | 주주 정보 |
| 보유주식수(지분율) | 양도 전 보유 주식수 및 지분율 |
| 양도일자 | 주식 양도 예정일 |
| 양도주식수(지분율) | 양도 주식수 및 지분율 |
| 수주식수(지분율) | 양수 주식수 및 지분율 |

제출 시기와 방법: 조세특례제한법 시행령 제116조의 34 제11항에 따라 사업재편계획서를 제출한다. 제출인과 확인자의 서명을 받아 세무서장에 제출한다.

### 작성요령 II - 사업재편계획이행보고서(기업 간 주식 등의 교환)

[별지 제70호의 18 서식] (2022. 3. 18. 개정)

#### 작성 목적

기업 간 주식 교환 계획이 계획대로 이행되었음을 보고하기 위한 서식이다.

#### 기재 항목

1. 대상법인 인적사항

사업재편계획서에 기재한 교환대상법인 및 교환양수법인의 인적사항을 동일하게 기재한다.

2. 주식 등의 양도·양수 명세

실제 이행 현황을 다음과 같이 기재한다.

| 항목 | 설명 |
|------|------|
| ©양도일 | 실제 주식 양도 일자 |
| ©양수주주 | 주식을 양수한 자의 성명 또는 법인명 |
| 생년월일 또는 사업자등록번호 | 양수주주 정보 |
| ©보유주식수(지분율) | 양수 후 보유 주식수 및 지분율 |
| ©1주당 시가 | 주식의 공정가액 |
| 발행주식수 | 양수 후 총 발행주식수 |
| ©양수주식수(지분율) | 실제 양수된 주식수 및 지분율 |
| 관계여부 | 양도인과 양수인 간의 특수관계 여부 |

법인양수도계약에 따라 교환대상법인의 주식을 인수하는 양수주주명, 양수주식수, 양수주주가 보유한 교환대상법인의 총주식수, 양도대상법인 또는 주주와 양수주주 간의 특수관계여부를 정확히 기재한다.

조세특례제한법 제121조의 30 제3항에 따른 적용 여부(제1호, 제2호, 제3호 해당 여부)를 별도로 표시한다.

제출 시기와 방법: 조세특례제한법 시행령 제116조의 34 제11항에 따라 사업재편계획이행보고서를 제출한다.

| ④생년월일 | 대표자 생년월일 |

2. 교환양수법인

| 항목 | 설명 |
|------|------|
| ⑥법인명 | 주식을 양수하는 법인 |
| ⑦사업자등록번호 | 법인의 사업자등록번호 |
| ⑧대표자성명 | 대표자명 |
| ⑨생년월일 | 대표자 생년월일 |

| ©수주식수(지분율) | 양수된 주식수 |

제출 시기와 방법: 조세특례제한법 시행령 제43조 제12항 또는 제116조의 34 제12항에 따라 주식등 양도·양수명세서를 제출한다. 세무서장에 제출한다.

### 작성요령 IV - 과세이연신청서

[별지 제40호의 4 서식] (2017. 3. 17. 개정)

#### 작성 목적

주식 교환에 따른 양도차익에 대해 과세를 이연하고자 할 때 신청하는 서식이다.

#### 기재 항목

1. 신청인 정보

| 항목 | 설명 |
|------|------|
| ①법인명 | 신청 법인명 |
| ②사업자등록번호 | 사업자등록번호 |
| ③대표자 성명 | 대표자명 |
| ④법인등록번호 | 법인등록번호 |
| ⑤사업장(본점)소재지 | 본점 주소 및 전화번호 |
| ⑥업종 | 업종 |
| ⑦사업개시일자 | 개시일 |
| ⑧사업연도 | 과세연도 |

2. 양도명세

| 항목 | 설명 |
|------|------|
| ⑨양도일자 | 주식 양도 날짜 |
