# 제4편 현금흐름표/자본변동표/이익잉여금처분계산서편

## 제1장 이익잉여금처분계산서의 의의

### 이익잉여금처분계산서의 정의와 목적

이익잉여금처분계산서는 기업의 이익잉여금이 어떻게 처분되었는지를 나타내는 계산서이다.

당해 연도의 명칭을 사용하는 것이 타당하다. 또한 직전연도는 결손금처리계산서를 작성하고 당해연도에는 이익잉여금처분계산서를 작성한 경우(반대의 경우 포함)에는 당해 연도를 기준으로 명칭을 사용하여야 하며, 이 경우의 작성양식은 이익잉여금처분계산서와 결손금처리계산서를 결합하여 사용하되 동일한 명칭에 결손금을 표시하는 경우(예: 미처분이익잉여금란에 미처리결손금 표시)에는 기호 부의 표시를 하여야 한다.

### 이익잉여금처분의 의결권

한편, 이익잉여금의 처분은 주주총회에서 확정된다. 다만, 예외적으로 중간배당 등은 이사회 결의로 할 수 있다. 따라서 주주총회에 제출되는 이익잉여금처분계산서는 보고기간말 현재 확정되지 아니한 처분안에 불과하므로 주주총회의 결의내용에 따라 변경될 수도 있다.

## 제2장 이익잉여금처분계산서의 체계

### 제1절 이익잉여금처분계산서의 계산구조

이익잉여금처분계산서는 미처분이익잉여금에서 출발하여 임의적립금 등의 이입액을 더하고 이익잉여금처분액을 차감하여 차기이월미처분이익잉여금을 구하는 방식으로 작성된다.

이 때 이익잉여금처분계산서상의 미처분이익잉여금은 다음과 같이 계산된다.

```
미처분이익잉여금 = 전기이월미처분이익잉여금 ± 회계정책변경누적효과
                ± 전기오류수정손익 - 중간배당액 ± 당기순손익
```

여기서 전기이월미처분이익잉여금은 전기의 이익잉여금처분계산서 상의 차기이월미처분이익잉여금과 같은 금액이다.

이렇게 미처분이익잉여금이 구해지면 차기이월미처분이익잉여금은 다음과 같이 계산된다.

```
차기이월미처분이익잉여금 = 미처분이익잉여금
                      + 임의적립금 등의 이입액
                      - 이익잉여금처분액
```

### 제3절 이익잉여금처분계산서의 과목과 범위

#### 1. 미처분이익잉여금

미처분이익잉여금은 회사의 영업활동에 의하여 이익잉여금이 발생하였으나, 특정목적에 사용하도록 처분되기 전의 잉여금으로서 전기이월미처분이익잉여금(또는 전기이월미처리결손금)에 회계정책의 변경으로 인한 누적효과(비교재무제표의 최초회계기간 직전까지의 누적효과), 중대한 전기 오류수정손익(비교재무제표의 최초회계기간 직전까지의 누적효과), 중간배당액 및 당기순손익(또는 당기순손실)을 가감한 금액이다(일반기준 2장 부록 실2.14).

##### (1) 전기이월미처분이익잉여금

전기이월미처분이익잉여금(또는 전기이월미처리결손금)은 직전기의 이익잉여금처분계산서(또는 결손금처리계산서)상의 차기이월미처분이익잉여금(또는 차기이월미처리결손금)과 같은 금액이다. 즉, 전기의 주주총회에서 미처분이익잉여금 중 이익잉여금처분 결의를 하고 남은 이익잉여금이 전기이월미처분이익잉여금이다.

##### (2) 회계정책변경누적효과

과거부터 변경된 새로운 회계처리기준을 자산 또는 부채의 해당 계정과목에 적용하여 계산된 손익의 누적효과를 비교재무제표 최초 회계기간의 미처분이익잉여금에 반영한다. 이에 대해서는 '재무상태표편 자본 제5장 제4절 회계변경과 오류수정'을 참조하기 바란다.

##### (3) 전기오류수정손익

이에 대해서는 '재무상태표편 자본 제5장 제4절 회계변경과 오류수정'을 참조하기 바란다.

##### (4) 중간배당액

상법 규정에 의하여 연 1회의 결산기를 정한 회사는 영업연도 중 1회에 한하여 이사회의 결의로 일정한 날을 정하여 그날의 주주에 대하여 배당을 할 수 있다(상법 462조의 3). 기중에 중간배당을 할 경우 중간배당액만큼을 미처분이익잉여금에서 상계하여야 한다.

##### (5) 당기순이익

당기순이익은 당기의 손익계산서상의 당기순이익과 같은 금액이다.

#### 2. 이익잉여금처분액

이익잉여금처분액은 당해 연도에 미처분이익잉여금에서 차감하여 각 적립금이나 배당금으로 사용한 금액이다. 주요 항목은 다음과 같다.

##### (1) 이익준비금

상법 상에서 현금배당을 할 경우 이익준비금을 적립하여야 한다(상법 462조). 이익준비금은 자본금의 50% 또는 준비금과 이익준비금 합계액이 자본금의 50%가 될 때까지 배당가능 이익의 10% 이상을 적립한다.

##### (2) 기타법정적립금

기타 관계 법령에 의하여 적립하여야 하는 기타법정적립금이 있는 경우 이를 처분액에 포함한다.

##### (3) 임의적립금

기업이 자발적으로 적립하고자 하는 임의적립금으로서 특정목적을 위해 또는 재무상태 개선 등을 위하여 설정한다.

##### (4) 배당금

주주에 대한 배당금으로서 통상 우선주배당금 및 보통주배당금으로 구분된다.

#### 3. 차기이월미처분이익잉여금

차기이월미처분이익잉여금은 미처분이익잉여금에 임의적립금 등의 이입액을 더하고 이익잉여금처분액을 차감한 금액이며, 당해 연도 다음 연도로 이월되는 미처분이익잉여금이다.

## 제3장 결손금처리계산서의 체계

### 제1절 결손금처리계산서의 개요

결손금처리계산서는 기업이 누적된 손실(결손금)을 처리하는 과정을 나타내는 계산서이다.

### 제2절 처리전결손금

처리전결손금은 전기이월미처리결손금에 회계정책변경누적효과, 전기오류수정손익 및 당기순손실을 가감한 금액이다.

#### 주요 구성요소

- **전기이월미처리결손금**: 직전기의 결손금처리계산서상의 차기이월미처리결손금과 같은 금액
- **회계정책변경누적효과**: 회계정책 변경으로 인한 누적효과
- **전기오류수정손익**: 전기 오류 수정으로 인한 손익
- **당기순손실**: 당기의 손익계산서상의 당기순손실 금액

### 제3절 결손금처리액

결손금처리액은 당해 연도에 처리전결손금에서 차감하여 각 적립금이나 자본에 이입되는 금액이다.

#### 주요 항목

##### (1) 임의적립금이입액

임의적립금 중에서 결손금 처리를 위해 이입하는 금액

##### (2) 기타법정적립금이입액

기타법정적립금 중에서 결손금 처리를 위해 이입하는 금액

##### (3) 이익준비금이입액

이익준비금을 결손금 처리에 이입하는 금액

##### (4) 자본잉여금이입액

자본잉여금을 결손금 처리에 이입하는 금액

### 제4절 차기이월미처리결손금

차기이월미처리결손금은 처리전결손금에서 결손금처리액을 차감한 금액이며, 다음 연도로 이월되는 결손금이다.

```
차기이월미처리결손금 = 처리전결손금 - 결손금처리액
```

### 제5절 결손금처리계산서의 양식 및 사례

#### 결손금처리계산서 표준 양식

| 항목 | 금액 |
|------|------|
| 처리전결손금 | |
| 임의적립금이입액 | |
| 기타법정적립금이입액 | |
| 이익준비금이입액 | |
| 자본잉여금이입액 | |
| 결손금처리액 | |
| 차기이월미처리결손금 | |

#### 결손금처리의 주요 특징

- 결손금 처리는 주주총회에서 확정된다
- 각 적립금이나 자본잉여금을 감소시켜 결손금을 처리한다
- 결손금 처리 후에도 남은 결손금은 차기에 이월된다
- 회계정책 변경이나 오류 수정이 있을 경우 이를 반영하여 표시해야 한다

## 요약

### 이익잉여금처분계산서의 구조
- **시작**: 미처분이익잉여금
- **더하기**: 임의적립금 등의 이입액
- **빼기**: 이익잉여금처분액(이익준비금, 기타법정적립금, 임의적립금, 배당금 등)
- **결과**: 차기이월미처분이익잉여금

### 결손금처리계산서의 구조
- **시작**: 처리전결손금
- **빼기**: 결손금처리액(적립금 이입액, 자본잉여금 이입액 등)
- **결과**: 차기이월미처리결손금

### 핵심 개념
- 이익잉여금의 처분은 주주총회에서 의결한다
- 중간배당은 예외적으로 이사회 결의로 할 수 있다
- 결손금 처리는 누적된 손실을 회사의 자본에서 상쇄하는 과정이다
- 회계정책 변경이나 오류 수정은 미처분/처리전 수치에 반영된다
