# 제2장 법인세 계산구조 및 소득처분

## 1. 법인세 계산구조

내국법인의 각 사업연도 소득에 대한 법인세는 소득금액 계산, 과세표준 계산, 세액 계산의 3단계 구조로 되어 있다.

### 〈1단계〉 소득금액 계산

| 항목 | 내용 |
|------|------|
| 익금총액 | 순자산증가액 (자본의 납입, 익금불산입항목 제외) |
| (−) 손금총액 | 순자산감소액 (자본의 환급, 잉여금의 처분, 손금불산입항목 제외) |
| = **각사업연도소득금액** | |

### 〈2단계〉 과세표준 계산

| 항목 | 내용 |
|------|------|
| 각사업연도소득금액 | |
| (−) 이월결손금 | 세무상 결손금을 이월하여 소득금액의 80%(중소기업 등은 100%)를 한도로 공제. 이월기간: '08.12.31. 이전 개시 사업연도 발생분 5년, '09.1.1.~'19.12.31. 개시 사업연도 발생분 10년, '20.1.1. 이후 개시 사업연도 발생분 15년 |
| (−) 비과세소득 | 공익신탁재산소득과 조특법상 비과세소득 |
| (−) 소득공제 | 유동화전문회사 등에 대한 소득공제와 조특법상 소득공제 |
| = **과세표준** | |

### 〈3단계〉 세액 계산

| 과세표준 | 세율 |
|---------|------|
| 2억원 이하 | 9%* |
| 2억원 초과 200억원 이하 | 19% |
| 200억원 초과 3,000억원 이하 | 21% |
| 3,000억원 초과 | 24% |

\* 성실신고확인대상 소규모법인은 과세표준 2억원 이하에 19% 세율 적용. 따라서 과세표준 200억원 이하에 대한 세율이 19%임.

| 항목 | 내용 |
|------|------|
| 법인세 산출세액 | 과세표준 × 세율 |
| (−) 감면공제세액 | 세액감면과 세액공제 |
| (+) 가산세 | 국세기본법상 가산세 및 법인세법상 가산세 |
| (+) 감면분추가납부세액 | 법인세법, 조특법 등에 따른 추가납부세액 |
| = 총부담세액 | |
| (−) 기납부세액 | 중간예납세액·원천납부세액·수시부과세액 |
| = **각 사업연도 소득에 대한 법인세 차감납부할 세액** | |
| (+) 토지 등 양도소득에 대한 법인세 | |
| (+) 미환류소득에 대한 법인세 | |
| = 차감납부할 세액 계 | |
| (−) 사실과 다른 회계처리 경정세액공제 | |
| (−) 분납할 세액 | |
| = **차감납부세액** | |

## 2. 세무조정

### 1. 개요

법인세 과세표준의 신고는 법인이 확정한 결산을 기초로 하여 신고하도록 되어 있다. 기업회계와 세무회계는 그 목적이 다르므로 기업회계에 따라 작성한 재무제표상의 당기순이익은 각 사업연도 소득과 차이가 있다. 따라서 당기순이익을 기초로 결산서와 세법의 차이를 조정하여 각 사업연도 소득을 계산하도록 하고 있다.

### 2. 세무조정

기업회계에서 수익에서 비용을 빼서 당기순이익을 계산하는 것과 같이 법인세법은 익금에서 손금을 빼서 각 사업연도 소득금액을 계산한다. 수익과 비용은 익금과 손금과 거의 같고 약간의 차이만 있으므로 당기순이익에 결산서와 세법의 차이를 가감하면 간편하게 각 사업연도 소득금액을 계산할 수 있다. 이를 세무조정이라고 한다.

| 결산서 | 세무조정 | 세법 |
|--------|---------|------|
| 수익 | (+) 익금산입 (−) 익금불산입 | = 익금 |
| (−) 비용 | (+) 손금산입 (−) 손금불산입 | = (−) 손금 |
| **당기순이익** | **(+) 익금산입 및 손금불산입, (−) 손금산입 및 익금불산입** | = **각사업연도소득금액** |

### 3. 세무조정항목

**(1) 가산조정**

① 익금산입: 익금항목을 결산상 수익으로 계상하지 않으면 당기순이익이 과소표시되므로 당기순이익에 과소표시된 금액을 더하는 조정

② 손금불산입: 손금불산입항목을 결산상 비용으로 계상하면 당기순이익이 과소표시되므로 당기순이익에 과소표시된 금액을 더하는 조정

**(2) 차감조정**

① 손금산입: 손금항목을 결산상 비용으로 계상하지 않으면 당기순이익이 과대표시되므로 당기순이익에서 과대표시된 금액을 빼는 조정

② 익금불산입: 익금불산입항목을 결산상 수익으로 계상하면 당기순이익이 과대표시되므로 당기순이익에서 과대표시된 금액을 빼는 조정

### 4. 세무조정계산서

세무조정의 내용은 세무조정계산서에 기재해야 한다. 주요 서식은 다음과 같다.

**(1) 소득금액조정합계표** (별지 제15호 서식): 모든 세무조정사항을 익금산입 및 손금불산입 란과 손금산입 및 익금불산입 란에 과목·금액·소득처분으로 구분하여 집계하는 서식. 다만, 기부금 한도초과액의 손금불산입과 전기기부금 한도초과이월액의 손금산입은 소득금액조정합계표에 기재하지 않고 직접 「법인세 과세표준 및 세액조정계산서」에 기재한다. 기부금 한도액은 소득금액을 한도로 하므로 소득금액조정합계표를 먼저 작성한 후에 기부금 한도액을 계산하기 위함이다.

**(2) 법인세 과세표준 및 세액조정계산서** (별지 제3호 서식): 당기순이익부터 시작해서 소득금액, 과세표준 및 세액의 계산내역을 기재하는 서식. 각 사업연도 소득에 대한 법인세는 물론이고 토지 등 양도소득에 대한 법인세와 미환류소득에 대한 법인세도 이 서식에 기재한다.

**(3) 자본금과 적립금조정명세서(을)** (별지 제50호 서식(을)): 유보로 소득처분된 금액을 관리하는 서식. 유보는 기업회계와 세무회계 간의 일시적 차이로서 그 후 반대 세무조정을 유발하므로 이 서식에서 관리한다.

**(4) 자본금과 적립금조정명세서(갑)** (별지 제50호 서식(갑)): 세무상 자본을 계산하는 서식.

> 세무상 자기자본 = 결산상 자기자본 + 유보잔액 − 손익 미계상 법인세비용

**(5) 소득자료[인정상여, 인정배당, 기타소득] 명세서** (별지 제55호 서식): 법인이 세무조정을 하면서 배당·상여·기타소득으로 소득처분한 경우에 소득자, 소득종류, 소득 귀속연도, 소득금액 등을 적어서 제출하는 서류. 국세청은 제출된 자료를 소득세 과세자료로 활용한다.

| 구분 | 관련서류 |
|------|---------|
| 법인이 신고하는 경우 | 「소득자료[인정상여, 인정배당, 기타소득] 명세서」를 세무서에 제출 |
| 국세청에서 결정·경정하는 경우 | 소득금액변동통지서로 법인에게 통지 |

### 5. 결산조정사항과 신고조정사항

#### (1) 세무조정에 따른 결산조정사항과 신고조정사항의 구분

익금산입, 손금불산입과 익금불산입 세무조정은 강제조정사항이므로 반드시 세무조정을 해야 한다. 그러나 손금산입 세무조정은 "결산조정사항"과 "신고조정사항"으로 나누어진다. 신고조정사항은 다시 임의조정사항과 강제조정사항으로 나누어진다.

#### (2) 결산조정사항

감가상각비, 충당금, 자산의 평가손실과 같은 비용은 외부와의 거래 없이 법인이 결산할 때 비용으로 계상하는 항목이다. 세법은 이러한 항목에 대하여 일정한 한도를 정하고 그 한도액 이내에서 비용으로 계상하면 그 비용 계상액을 손금으로 인정하고 한도미달액에 대해서는 손금산입의 세무조정을 허용하지 않는다. 이를 결산조정사항이라고 한다.

**결산조정사항 목록**:

| 구분 | 결산조정사항 | 예외 (신고조정 가능) |
|------|------------|-------------------|
| 자산의 감가상각 | ① 감가상각비 | K-IFRS 적용법인의 유형자산·비한정 무형자산(임의조정), 조특법 §28·§28의2·§28의3·§28의4에 따른 설비투자자산(임의조정), 2016.1.1. 이후 취득 업무용승용차(강제조정), 감면사업 감가상각의제(강제조정), 특수관계인 양수 자산 미달액(강제조정) |
| | ② 대손금 | 소멸시효 완성 등 법적 소멸시 강제조정 |
| | ③ 재고자산 평가손실 | |
| | ④ 주식의 평가손실 | |
| | ⑤ 천재지변 등 유형자산 평가손실 | |
| | ⑥ 시설개체·기술낙후 생산설비 폐기손실 | |
| 충당금 | ⑦ 퇴직급여충당금 | 퇴직연금충당금은 강제조정 |
| | ⑧ 대손충당금 | |
| | ⑨ 구상채권상각충당금 | |
| 준비금 | ⑩ 법인세법상 준비금 (책임준비금, 비상위험준비금, 고유목적사업준비금) | 회계감사대상법인의 고유목적사업준비금(임의조정), K-IFRS 적용법인의 비상위험준비금(임의조정) |

> **[사례] 결산조정사항**
> 대손충당금 한도액 100, 설정액이 60인 경우: 세무조정 없음 (한도미달액 40을 손금산입하면 안 됨). 손금인정액 60.
> 대손충당금 한도액 100, 설정액이 140인 경우: 손금불산입 40. 손금인정액 100.

#### (3) 신고조정사항 중 임의조정사항

일시상각충당금과 조세특례제한법상 준비금과 같이 기업회계상 비용으로 인정되지 않는 것을 조세정책적 목적에서 과세이연혜택을 주기 위해 세무상 손금으로 인정하는 항목. 기업회계기준에 위배되는 항목이므로 결산상 비용으로 계상하지 않더라도 손금산입의 세무조정을 할 수 있도록 허용한다.

**임의조정사항**: K-IFRS 적용법인의 유형자산·비한정 무형자산 감가상각비, 조특법상 설비투자자산 감가상각비, 일시상각충당금과 압축기장충당금, 비상위험준비금, 해약환급금준비금, 고유목적사업준비금, 조세특례제한법상 준비금

> **[사례] 임의조정사항**
> 일시상각충당금 한도액 100, 설정액이 60인 경우(세부담 최소화 가정): 손금산입 40. 손금인정액 100.
> 일시상각충당금 한도액 100, 설정액이 140인 경우: 손금불산입 40. 손금인정액 100.

#### (4) 신고조정사항 중 강제조정사항

인건비, 광고선전비, 세금과공과와 같은 항목은 외부와의 거래에 따라 손금귀속시기와 손금액이 확정된다. 결산조정사항과 임의조정사항이 아닌 항목은 모두 강제조정사항이다.

#### (5) 결산조정사항과 신고조정사항의 차이

| 구분 | 결산조정사항 | 신고조정사항 |
|------|------------|------------|
| 손금공제방법 | 결산상 비용으로 회계처리하는 방법만 인정 | 결산상 비용으로 회계처리하는 방법과 세무조정하는 방법 모두 인정 |
| 손금귀속시기 | 요건을 구비하고 결산상 회계처리한 사업연도 | 요건을 구비한 사업연도 |
| 누락된 경우 | 세무조정 및 경정청구할 수 없음 | 세무조정 및 경정청구할 수 있음 |

#### (6) 임의조정사항을 손금산입하지 못한 경우 경정청구 가능 여부

과세당국, 조세심판원, 법원 모두 신고조정이 가능한 것으로 해석하고 있다.

- **일시상각충당금**: 수령한 날이 속하는 사업연도에 손금산입하지 아니한 경우 경정청구기간 내에 경정청구를 통하여 손금산입할 수 있다(조심 2011중1984, 2012. 6. 8.).
- **조특법상 준비금**: 적용 요건을 충족하였으나 적용하지 아니하고 신고한 경우 임시주주총회에서 이익잉여금 재처분을 의결하고 재무제표를 수정하여 준비금을 적립한 경우 수정신고 또는 경정청구 기한 이내에 할 수 있다(기획재정부 조세특례제도과-236, 2010. 3. 9.).
- **K-IFRS 적용법인의 감가상각비**: 감가상각비 신고조정 특례규정을 적용하지 않고 신고한 경우 경정청구를 통하여 적용할 수 있다(사전-2016-법령해석법인-0337, 2016. 9. 29.).

### 6. 잉여금처분에 의한 신고조정

#### (1) 잉여금처분에 의한 신고조정대상

임의조정사항 중에서 구상채권상각충당금, 비상위험준비금, 해약환급금준비금, 고유목적사업준비금과 조세특례제한법상 준비금을 신고조정하는 경우에는 반드시 해당 사업연도의 이익처분에 있어서 그 과목의 적립금을 적립하고 손금산입의 세무조정을 해야 한다(법법 §28②, §31②, §32, §35②, §61①).

| 구분 | 신고조정 대상 법인 |
|------|-----------------|
| 충당금: 구상채권상각충당금 | K-IFRS 적용 법인 |
| 준비금: 비상위험준비금 | K-IFRS 적용 법인 |
| 준비금: 고유목적사업준비금 | 회계감사 대상 비영리내국법인 |
| 준비금: 해약환급금준비금 | 보험회사 |
| 조세특례제한법상 준비금 | 준비금 설정대상 법인 |

#### (2) 잉여금처분에 의한 신고조정방법

**1) 손금산입**: 해당 사업연도의 정기주주총회 등에서 잉여금을 처분하여 적립금을 적립한 경우 손비로 계상한 것으로 보아 손금에 산입한다(법법 §61①). 처분가능이익이 부족한 경우 처분가능이익을 한도로 적립하되, 준비금은 한도액까지 손금에 산입할 수 있다. 부족적립액은 그 후 사업연도에 처분가능이익이 발생하면 추가 적립해야 한다(법기통 61-98...1).

**2) 익금산입**: 준비금을 익금에 산입하는 경우에는 동 적립금을 이익잉여금으로 이입하고 준비금을 익금산입하여야 한다. 과소이입시 이입액에 관계없이 환입해야 할 금액을 익금에 산입한다. 과다이입시 준비금을 설정한 사업연도에 적립금을 적립하지 않은 것으로 보아 손금산입한 사업연도에 준비금을 손금불산입한다(법령 §98①).
