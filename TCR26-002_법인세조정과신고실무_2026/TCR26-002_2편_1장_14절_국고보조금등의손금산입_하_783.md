# Step II 서식의 이해

## 작성요령 I - (국고보조금등/공사부담금/보험차익) 상당액 손금산입조정명세서

### [별지 제35호 서식] (2021.10.28. 개정)

본 서식은 국고보조금등, 공사부담금, 보험차익 중 해당 항목을 선택하여 작성한다.

**1. 국고보조금등으로 취득한 자산 손금산입액 조정**

| 항목 | 내용 |
|------|------|
| ① 보조금 수령액 | 수령한 국고보조금등의 금액 |
| ② 계정과목(과목명, 코드) | 표준재무상태표 [별지 제3호의2 서식(1), (3)]의 계정과목명과 코드를 적는다 |
| ③ 취득한 자산가액 | 국고보조금등으로 이미 취득한 자산가액 |
| ④ 취득할 자산가액 | 사용계획서 제출금액으로 한다 |
| ⑤ 계(③+④) | 취득한 자산가액과 취득할 자산가액의 합계 |
| ⑥ 해당 자산에 사용되는 국고보조금등 | Min(①, ⑤) |
| ⑦ 용인한도액(⑦=⑥) | ⑥과 동일 |
| ⑧ 회사계상액 | 회사가 손금산입한 금액 |
| ⑨ 한도초과액(⑧-⑦) | 회사계상액이 한도를 초과한 금액 |

**2. 공사부담금으로 취득한 자산 손금산입액 조정**

| 항목 | 내용 |
|------|------|
| ⑩ 공사부담금 가액 | 제공받은 공사부담금의 금액 |
| ⑪ 계정과목(과목명, 코드) | 취득한 자산의 계정과목명과 코드 |
| ⑫ 취득한 자산가액 | 공사부담금으로 이미 취득한 자산가액 |
| ⑬ 취득할 자산가액 | 사용계획서 제출금액 |
| ⑭ 지급받은 자산가액 | 수요자 등으로부터 직접 제공받은 자산의 가액 |
| ⑮ 계(⑫+⑬+⑭) | 합계 |
| ⑯ 용인한도액(⑯=⑩) | 공사부담금 가액과 동일 |
| ⑰ 회사계상액 | 회사가 손금산입한 금액 |
| ⑱ 한도초과액(⑰-⑯) | 회사계상액이 한도를 초과한 금액 |

**3. 보험차익으로 취득한 자산 손금산입액 조정**

| 항목 | 내용 |
|------|------|
| ⑲ 보험금 | 지급받은 보험금 |
| ⑳ 멸실손괴 자산가액 | 멸실 또는 손괴된 자산의 장부가액 |
| ㉑ 보험차익(⑲-⑳) | 보험금에서 장부가액을 차감한 금액 |
| ㉒ 계정과목(과목명, 코드) | 대체 취득한 자산의 계정과목 |
| ㉓ 대체한 자산가액 | 이미 대체 취득한 자산가액 |
| ㉔ 대체할 자산가액 | 사용계획서 제출금액 |
| ㉕ 계(㉓+㉔) | 합계 |
| ㉖ 대체자산에 사용되는 보험차익 | Min(㉑, ㉕-⑳) |
| ㉗ 용인한도액(㉗=㉖) | ㉖과 동일 |
| ㉘ 회사계상액 | 회사가 손금산입한 금액 |
| ㉙ 한도초과액(㉘-㉗) | 회사계상액이 한도를 초과한 금액 |

**4. 미사용분 익금산입액 조정**

사용계획서 제출에 의한 손금산입액 중 기한 내 미사용 또는 해산 등에 의한 익금산입액을 아래 구분코드(⑩)에 따라 계산한다.

| 구분 | 국고보조금 | 공사부담금 | 보험차익 |
|------|-----------|-----------|---------|
| 코드 | 1 | 2 | 3 |

- 「⑪ 원래 손금산입기준액」란은 원래 손금산입 사업연도의 손금산입기준금액 ⑥, ⑯, ㉖란의 각 금액을 적고, 「⑫ 원래 손금산입액」란은 원래 손금산입 사업연도의 손금산입액 ⑦, ⑯, ㉗란의 각 금액을 적되, 원래 한도초과액은 제외한다.

## 작성요령 II - 보조금 등 수취명세서

### [별지 제35호의2 서식] (2004.3.5. 신설)

보조금 등 수취명세서는 법인이 국고보조금, 공사부담금, 보험금 등을 수취한 경우 그 내역을 기재하여 법인세 과세표준 신고시 첨부하는 서식이다.

|---------|

# Step III 사례와 서식작성실무

## 예제: 보험차익의 손금산입과 서식 작성

다음 자료에 의하여 (주)삼일의 회계처리와 세무조정을 예시하고, 손금산입조정명세서를 작성하시오(단, 일시상각충당금의 손금산입방법은 신고조정에 의함).

**기본 자료**

1. 사업연도: 2025.1.1.~12.31.
2. 2025.5.1. 화재가 발생하여 취득원가 5억원, 감가상각누계액 2억원(상각부인액 0.5억원)인 건물이 전소되었다.
3. 2025.9.30. 위 건물에 대한 보험금을 4억원 수령하였다.
4. 2025.12.1. 위 건물과 같은 용도의 건물을 8억원에 취득하였다. 회사는 동 건물에 대한 감가상각비를 0.6억원 계상하였으나, 세무상 감가상각비 한도액은 0.5억원이다.

---

### 1. 회계처리와 세무조정

**① 화재발생시(2025.5.1.)**

> (차) 감가상각누계액 200,000,000, 재해손실 300,000,000 / (대) 건물 500,000,000

**② 보험금 수령시(2025.9.30.)**

> (차) 현금 400,000,000 / (대) 보험금수익 400,000,000

세무조정:

- 〈익금불산입〉 감가상각누계액(상각부인액 추인) 50,000,000 (△유보)
- 〈손금산입〉 일시상각충당금 50,000,000 (△유보)
- (*1) 세무상 보험차익 = 보험금수령액 - 멸실자산의 장부가액 = 4억원 - (5억원 - 2억원 + 0.5억원) = 0.5억원
- 손금산입한도액 = Min[① 0.5억원, ② 8억원 - (5억원 - 2억원 + 0.5억원)] = 0.5억원

**③ 건물구입시(2025.12.1.)**

> (차) 건물 800,000,000 / (대) 현금 800,000,000

**④ 결산시(2025.12.31.)**

> (차) 감가상각비 60,000,000 / (대) 감가상각누계액 60,000,000

세무조정:

- 〈손금불산입〉 감가상각누계액(한도초과액) 10,000,000 (유보)
- 〈손금불산입〉 일시상각충당금 3,125,000 (유보)
- (*2) 50,000,000 × 0.5억원 / 8억원 = 3,125,000원

### 2. 보험차익상당액 손금산입조정명세서 [별지 제35호 서식]

**(주)삼일, 사업연도 2025.1.1.~2025.12.31.**

**3. 보험차익으로 취득한 자산 손금산입액 조정**

| 항목 | 금액 |
|------|------|
| ⑲ 보험금 | 400,000,000 |
| ⑳ 멸실손괴 자산가액 | 350,000,000 |
| ㉑ 보험차익(⑲-⑳) | 50,000,000 |
| ㉒ 계정과목 | 건물 |
| ㉓ 대체한 자산가액 | 800,000,000 |
| ㉔ 대체할 자산가액 | - |
| ㉕ 계(㉓+㉔) | 800,000,000 |
| ㉖ 대체자산에 사용되는 보험차익 | 50,000,000 |
| ㉗ 용인한도액(=㉖) | 50,000,000 |
| ㉘ 회사계상액 | 50,000,000 |
| ㉙ 한도초과액(㉘-㉗) | 0 |

- 멸실손괴 자산가액(⑳)은 세무상 장부가액 = 5억원 - 2억원 + 0.5억원(상각부인액) = 3.5억원
- 보험차익(㉑) = 4억원 - 3.5억원 = 0.5억원
